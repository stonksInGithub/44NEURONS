Disease detection using Machine Learning

Pneumonia detector

Abstract: Detection of pneumonia through x ray images using Conventional Neural Network, trained with over 1300 images with an average accuracy of 96%.

Introduction: Pneumonia is an infection that inflames air sacs in one or both lungs, which may fill with fluid. Through the x ray images of the chest and with the help of convolutional neural network and modern machine learning technology, anywhere anytime one can diagnose themselves. The CNN model is trained with over 1300 perfectly classified images and has an average accuracy of 96%.
Aim of this project is to help people diagnose themselves anytime anywhere. Though it has a good accuracy it is to be noted that this is not a completely accurate information and one must consult an appropriate doctor for confirmation.


Design: This project is programmed in python with the back-end being tensorflow. We used flask to integrate the model and the UI.

Data: The testing data used is from Kaggle. The dataset is organized into 3 folders (train, test, val) and contains subfolders for each image category (Pneumonia/Normal). There are 5,863 X-Ray images (JPEG) of 2 categories (Pneumonia/Normal). The data provided is very precise and accurate

Model: This project uses keras' VGG16 model which is 16 layers deep. The model pre-processes the given chest x-ray dataset and divides it into two splits namely testing and training. Then the model is initialised with the following splits and the model is added with activation functions. Then the model is flattened. We now have the fully connected neural network following which the model is compiled with adam optimizer.During the training of the model, it is noted that it has an average accuracy of 96% and during the validation it has an average accuracy of 92%
